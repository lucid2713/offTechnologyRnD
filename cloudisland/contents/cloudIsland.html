Cloud Island





Cloud Island (work in progress) is a virtual land which represents the digitized world where we are living and heading for. Unlike the name ‘cloud’ and general image about the future IT world, the Cloud Island focuses on showing the infrastructures and machines - the physical parts of the digitals - that enable our ‘cloud-based digital world’.

The Cloud Island is an interactive platform which looks like a virtual game, but in fact, it is an interactive visual report about our digitized society. It aims to reveal how the 0 and 1(information technology) entangles with the actual biogeochemical environment.

Cloud Island will be realized with a 3D modeling program, and relevant real data will also be implemented.
The visual aspect of the idea is inspired by city/urban planning simulation games such as Simcity/Sims/Minecraft.
The features and data which shape the island are going to be represented and based on the North Netherlands area.

Through the cloud island, artist Mina wants to show the collected information and assembled data about the infrastructure of the IT world in a playable and accessible way.
